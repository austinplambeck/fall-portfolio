Projects

Of the Sky: The Game

A choice-based text game that utilizes JavaScript objects, enabling the player to make choices and traverse through the game. The game branches into various subplots, ultimately leading to two different endings. It features a main menu, an exit menu pop-up, and toggleable music.

To-do List App

A to-do list web app that allows the user to add new to-do items, check off to-dos, and delete to-dos. I added a sorting feature, organizing the to-dos by "Completed" & "Incomplete." In addition, the to-dos save to local storage, allowing the user to pick up where they left off even after closing the web app.

Random Film Generator

A random film generator I built for the "Mixed Messages" Project on Codecademy. Initially only run in Node.js, I expanded the project and made a web app out of it, adding two different settings for generating films: sort by genre and by all. Currently building Version 2.0 (utilizing the IMDb API).

Typewriter App
IN PROGRESS

A Google Docs clone, but made for writers. Featuring typewriter sounds and changable backgrounds and color schemes. The user can set a word-count goal and watch the progress bar go up as they write. Inputted text can be copied or saved as a PDF. The text will also save to local storage in case the web app crashes.
